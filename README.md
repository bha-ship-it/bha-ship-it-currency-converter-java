# Currency Converter

A Java console application that converts one currency into another using predefined exchange rates.

## Features

- Convert between INR, USD, and EUR
- User selects base currency
- User selects target currency
- Accepts amount from user
- Displays converted value clearly

## Supported Currencies

1. INR - Indian Rupee
2. USD - US Dollar
3. EUR - Euro

## Sample Exchange Rates

- 1 USD = 83 INR
- 1 EUR = 90 INR

## How to Run

```bash
javac CurrencyConverter.java
java CurrencyConverter
```
